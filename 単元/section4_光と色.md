# 【Section 4】光と色

公式テキスト（32〜44ページ）の内容に基づき、色の様相、照明の性質、および混色の法則についてまとめます。

---

## 1. 心理的な見えによる色の分類
色は、色相・明度・彩度の3属性（またはXYZなどの数値）が同じでも、現れ方によって見え方の印象が大きく変わります。これを**色の様相（色のモード）**の違いと呼びます。

### カッツ (Katz, D.) の分類（心理学的分類）
ドイツの心理学者カッツは、色の現れ方を以下のように分類しました（JIS Z 8105でも定義されています）。

1.  **面色 (フィルムカラー / JIS: 開口色):** 青空のように、質感や陰影がなく距離感が不明確で、色だけがそこに見えているような状態。補色残像もこれにあたる。
2.  **表面色 (サーフェスカラー):** 物体の表面の色。質感や陰影があり、自分との明確な距離感や存在感を感じる。我々が見る物体の色の多くはこれ。
3.  **透明面色 (トランスペアレントフィルムカラー):** 色ガラスやセロハンを通して景色を見たときの色。透明感があり、景色に色がくっついているように見える。
4.  **鏡映色 (ミラードカラー):** ミラーボールや赤い鏡に映った色のように、鏡面に映り込んだ景色の色の見え方。
5.  **容積色 (ボリュームカラー / 空間色):** 色のついた水（メロンソーダなど）が透明な瓶に入っているような状態。奥行きや厚み、空間を色が満たしている感じがある。
6.  **光沢 (ラスター / JIS: グロス):** 物体の表面が白く光った状態（ハイライト部分）の色の見え方。
7.  **光輝 (ルミノシティ / JIS: 発光色・知覚色):** ろうそくの炎（絵画に描かれた炎など）のように、光を発して輝いているように感じられる色の見え方。※テレビ画面などは物体のように見えるため光輝とは言わない。
8.  **灼熱 (グロー):** 焼けた鉄のように、表面だけでなく内部からも光が満ち溢れて輝いているような見え方。

### 測色学的分類
色を発する側からの分類で、知覚色と心理物理色に分けられます。
*   **知覚色:** 人間の視覚を通して知覚された色。物体色に対応し、マンセル表色系（顕色系）などで表される。
*   **心理物理色:** 物理的に測定して規定された色。光源色に対応し、XYZ表色系（混色系）などで表される。

## 2. 照明の性質を表すもの
光の明るさを定量的に測ることを**測光**といい、測られた量を**測光量**と呼びます。
測光量（心理物理量）は、物理的なエネルギー量である「放射量」に、人間の明るさに対する感度「標準分光視感効率」を掛け合わせて求められます。

### 測光量の4つの指標
1.  **光束 (Luminous flux):**
    *   光源から全方向に放射される、単位時間あたりの光の量（光源そのものの明るさ）。
    *   単位: **ルーメン (lm)**
2.  **光度 (Luminous intensity):**
    *   光源からある1方向に放射される光の強さ（点光源からの光の強さ）。
    *   単位: **カンデラ (cd)** ※語源はキャンドル（蝋燭）。
3.  **輝度 (Luminance):**
    *   光源の輝きや、照明された物体の反射面が「どれだけ明るいか」を表す（単位面積あたりの光度）。光源が小さいほど輝度が高くなり、眩しさ（グレア）を生じる。
    *   単位: **カンデラ毎平方メートル ($cd/m^2$)**
4.  **照度 (Illuminance):**
    *   照射面（照らされた面）に入射する光束の大きさ。
    *   単位: **ルクス (lx)**

### 色温度 (Color temperature)
*   **黒体（完全放射体）:** 外部からの電磁波をすべて吸収し、反射しない架空の物体。
*   **プランクの法則:** 黒体の温度が上がると光を発し、その温度によって分光分布（光の色）が変化する。
*   光源の色を、この黒体の温度（**ケルビン: K**）に当てはめて表したものが色温度。
    *   *例:* 白熱電球(2800K)、蛍光ランプ昼光色(6500K)

### 測色の照明（標準イルミナント）
色を測定する際の基準となる光（分光分布の数値データ）をCIEが規定しています。
*   **CIE標準イルミナント $D_{65}$:** 色温度約6500Kの平均的な自然昼光。紫外域を含み、可視域の各波長を平均的に含む。（実際の電球＝標準光源は存在しない）
*   **CIE標準イルミナント A:** 色温度約2800Kの白熱電球の光。長波長（赤）を多く含む。（実際の電球＝標準光源Aが存在する）

## 3. 混色（グラスマンの法則）
ドイツの数学者グラスマン(Grassmann)が1853年にまとめた、加法混色における法則性です。これがXYZ表色系の基礎となっています。

### 等色実験とメタメリズム
*   **等色:** ある色に対して、RGB（赤・緑・青）の3つの光の強さを調整し、全く同じ色に見えるようにすること。
*   **条件等色（メタメリズム）:** 分光分布が異なる光同士が、同じ色に見える現象。（人間の眼の錐体の反応が同じになれば区別できないため起こる）

### 加法混色の法則性
1.  **色の三色性:**
    *   独立した3つの色刺激（原刺激：R, G, B）を適切な比率で混色すれば、すべての色を作り出すことができる。
    *   *等色式:* $[C] \equiv R[R] + G[G] + B[B]$ （$\equiv$ は等色を表す合同記号）
2.  **色の等価性:**
    *   等色している2つの色に対して、それぞれ同じ色を加えても、混色結果はやはり等色する。
3.  **色の加法性:**
    *   等色式同士は足し算ができる（レシピの足し算）。
    *   *例:* $[C_1] \equiv R_1[R] + G_1[G]$ と $[C_2] \equiv G_2[G] + B_2[B]$ を足すと、$[C_1 + C_2] \equiv R_1[R] + (G_1+G_2)[G] + B_2[B]$ となる。
4.  **色の連続性:**
    *   混色する色光の量を連続的に変化させると、出来上がる混色結果も連続的に変化する。
