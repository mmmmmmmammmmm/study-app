【Section 2】色彩調和論

公式テキスト（24〜30ページ）の内容に基づき、代表的な色彩調和論の特徴と原理をまとめます。

1. ジャッド (Judd, D.B.) の4つの原理

アメリカの色彩学者ジャッドは、古今東西のあらゆる色彩調和論を読破し、共通する基本原理を以下の4つにまとめました。

  1. 秩序の原理:
    • 知覚的に等間隔な尺度で構成された色空間（色立体や色相環）から、規則的（直線や正三角形など単純な幾何学図形）に選ばれた配色は調和する。
    • 例: 補色色相配色（色相環上で直線で結ばれる2色）。オストワルトの調和論もこれに属する。
  2. なじみの原理（親近性の原理）:
    • 人々が自然界や日常で慣れ親しんでいる色の配列は調和する。
    • 例: 夕焼け空のグラデーション、ナチュラルハーモニー。
  3. 類似性の原理（共通性の原理）:
    • 共通要素を持つ色の集まりは、その共通性によって調和する。
    • 例: 同一色相配色、同一トーン配色。
  4. 明瞭性の原理:
    • 色の差が小さすぎたり意図が曖昧な配色は調和せず、はっきりとした対比（コントラスト）がある配色は調和する。
    • 例: ビコロール配色、トリコロール配色。

2. シュヴルール (Chevreul, M.E.) の色彩調和論

フランスの化学者・ゴブラン織物工場長のシュヴルールは、1839年に『色彩の同時対比の法則とこの法則に基づく配色について』を著し、印象派や新印象派に大きな影響を与えました。彼は色彩調和を「2種類・6通り」に分類しました。

A. 類似色の調和（3通り）

  1. 単一色相における異なる色調の調和: 同一色相で明度・彩度が異なる配色。（例: 同一色相の濃淡）
  2. 隣接近似色相による色調類似の調和: 色相が隣接し、トーン（色調）が同一または類似の配色。
  3. 一つの主張色による調和: 全ての色を色ガラスを通して見たような、全体が同一色相に支配された配色（ドミナントカラー）。

B. 対比の調和（3通り）

  1. 同一色相による色調対比の調和: 同一色相で、明暗や強弱の差が大きく対比される配色。（例: ペールトーンとダークトーン）
  2. 隣接色相における色調対比の調和: 隣接色相で、明度差やトーンの差が大きい配色。
  3. 色相対比を増大するように選ばれた色彩対比の調和: 色相が大きく離れた配色。（例: 明度差のある補色配色）

3. ルード (Rood, O.N.) の色彩調和論

アメリカのオグデン・ルードは、1879年に『現代色彩学』を著しました。

  • 色相の自然連鎖 (ナチュラルハーモニー):
    • 自然界の光の当たり方に基づき、「黄色寄りの色の明度を高く、青紫寄りの色の明度を低くする」配色が調和すると述べました。
    • ※「ナチュラルハーモニー」という名称は後から付けられたものです。PCCSの色相環もこの原理に則って配置されています。

4. オストワルト (Ostwald, W.) の色彩調和論

ドイツの化学者（ノーベル賞受賞者）オストワルトは、1917年に独自の表色系を発表しました。

  • 「調和は秩序に等しい」:
    • 彼の調和論はジャッドの「秩序の原理」に分類されます。
    • 配色の各色の間に規則的な関係があることが調和を生むと考えました。
  • 等色相三角形における調和:
    • 等白系列: 白色量(W)が共通する色の配列。
    • 等黒系列: 黒色量(B)が共通する色の配列。
    • 等純系列 (シャドウシリーズ): 純色量(C)と白色量(W)の比率が一定の色の配列（キアロスクーロの一種と考えられ、なじみの原理にもあたる）。

5. イッテン (Itten, J.) の色彩調和論

スイスの美術教育者であり、バウハウスの教授であったヨハネス・イッテンは、『色彩の芸術』で調和論を展開しました。

  • 12色相環: 絵具の三原色（赤・黄・青）を基に、独自の12色相環を作成。向かい合う色は補色となります。
  • 調和のとれた面積比:
    • 「調和には客観性が必要であり、調和は力の均衡である」とし、色の明度の逆数を面積比にすることで調和が取れると提唱しました。
    • 比率: 黄(3) : 橙(4) : 赤(6) : 緑(6) : 青(8) : 紫(9)
    • （明るい黄色は面積を小さく、暗い紫は面積を大きくする）

6. ムーンとスペンサー (Moon, P. & Spencer, D.E.) の色彩調和論

1944年に連名で発表された論文で、色彩調和を「数値化・法則化」したことで有名です。

  • 調和の2条件:
    1. 2つの色の差が曖昧でないこと。
    2. 独自の「オメガ空間」内で、色が単純な幾何学図形で関係付けられること。
  • 3つの調和関係と不調和:
    • 調和する関係を 「同一」「類似」「対比」 の3つに限定し、それぞれ色相差・明度差・彩度差の範囲を設定。
    • これに当てはまらない曖昧な差を「第1不明瞭」「第2不明瞭」とし、不調和であると定義しました。
  • 美度 (M) の計算:
    • $M = 秩序(O) / 複雑さ(C)$
    • この公式で評価を数値化し、美度が 0.5以上 であれば調和するとしました。
