【Section 4】色の表示 (XYZ表色系と均等色空間)

公式テキストに基づき、混色系の代表である「CIE 1931 XYZ表色系」と、知覚的な色の差を表現する「均等色空間 ($L^a^b^*$表色系など)」についてまとめます。

1. XYZ表色系の基本

XYZ表色系は、グラスマンの「色の三色性」の原理（加法混色）に基づき、物体色も光源色もすべての色を数値で正確に表示できる表色系です。

原刺激 (X, Y, Z) と虚色

  • 実際の光の三原色(R, G, B)による等色実験では、鮮やかな青緑など「マイナスの値」を用いないと再現できない色が存在します。
  • 計算を簡便にするため、すべての色を「プラスの値」で表せるように、現実には存在しない光（虚色）を三原色として設定しました。
    • $[X]$ : 赤みだけを持つ原刺激
    • $[Y]$ : 緑みと「明るさ」を持つ原刺激
    • $[Z]$ : 青みだけを持つ原刺激

等色式と三刺激値

  • 等色式: $[C] \equiv X[X] + Y[Y] + Z[Z]$
  • 三刺激値 ($X, Y, Z$): 各原刺激を混色する「量（大さじ〇杯のような割合）」。大文字の斜体で表されます。とくに $Y$ の値は色の「明るさ（反射率・透過率）」 を表します。

CIE標準観測者と等色関数

  • 色の見え方は人によって異なるため、CIEは標準的な目の感度を「等色関数 ( $\bar{x}(\lambda), \bar{y}(\lambda), \bar{z}(\lambda)$ )」として数値化しました。
  • 2度視野: CIE 1931 測色標準観測者（中心窩の感度）。
  • 10度視野: CIE 1964 測色補助標準観測者（より広い範囲を見る場合の感度）。

2. 色度座標と $xy$ 色度図

三刺激値($X, Y, Z$)のままでは「量」であるため、どんな色味なのか直感的に分かりません。そこで、割合（パーセンテージ）に変換して色味（色相と彩度）を表します。

色度座標 ($x, y$)

  • $x = X / (X+Y+Z)$
  • $y = Y / (X+Y+Z)$
  • これらを色度座標（小文字の斜体）と呼びます。（※ $x+y+z=1$ になるため、$z$ は省略されます）

$xy$ 色度図の読み方

色度座標 $x$ と $y$ を平面グラフにしたものが $xy$色度図 です。
* $x$軸: 大きくなるほど赤みが増す。
* $y$軸: 大きくなるほど緑みが増す。
* 左下: 青みが増す。
* 白色点: 中央付近 ($x=0.333, y=0.333$)。ここに近づくほど彩度が低くなる。
* スペクトル軌跡: 図の外側の馬蹄形（アイロン型）のカーブ。各波長の単色光（最も彩度が高い色）が並ぶ。
* 純紫軌跡: スペクトル軌跡の両端を直線で結んだ部分。単波長と長波長の混色で作られる紫が並ぶ。
* 色再現域: テレビ(RGB)や印刷(CMY)などが出力できる色の範囲。色度図の中に三角形や多角形で示され、印刷の方がテレビより色再現域が狭い。

3. 均等色空間 (UCS) と $L^a^b^*$

XYZ表色系（$xy$色度図）では、「グラフ上の距離」と「人間の見た目の色の差」が一致しないという欠点がありました。

マクアダム楕円

  • マクアダムは、$xy$色度図上で「同じ色に見える範囲」を調べました。その結果、緑の領域では範囲（楕円）が非常に大きく、青の領域では小さくなるなど、図上の距離と知覚的な差に大きなズレがあることが判明しました。この楕円をマクアダム楕円（マクアダムの偏差長円）と呼びます。

CIE $L^a^b^*$ 色空間 (エルスター・エースター・ビースター)

知覚的な色の差が、空間内の距離とほぼ一致するように変換された3次元の空間が均等色空間 (UCS: Uniform Color Space) です。
* $L^*$ (明度): 縦軸。0（最も暗い）から100（最も明るい）で表す。（マンセル明度の約10倍に相当）。
* $a^$, $b^$ (色度座標):
* $+a^$: 赤方向 ／ $-a^$: 緑方向
* $+b^$: 黄方向 ／ $-b^$: 青方向

色差 ($\Delta E^*_{ab}$)

2つの色（ターゲットとサンプル）の空間的な距離を計算することで、色の差（色差）を数値化します。
* 産業界では、製品の色のバラツキ許容範囲（許容色差）が設定されています。例えば厳密な工業用塗料の調色では、色差 $\Delta E^*_{ab}$ は 0.4〜0.8 程度と非常に厳しく管理されます。

4. オストワルト表色系 (Ostwald Color System)

ドイツの化学者オストワルトが1917年に発表。あらゆる色は「白・黒・純色」の混合で作られるという考えに基づいています。

基本原理

  • 混合比: 白色量($W$) + 黒色量($B$ / $s$) + 純色量($C$ / $p$) = 100
  • 回転混色機を用いた実験結果を基に体系化されています。

色相環

  • ヘリングの反対色説（心理四原色：赤・黄・緑・青）を基本とした8色相。
  • 各色相を3分割し、合計24色相。
  • 向かい合う色は補色の関係にあります。

無彩色の段階と記号

  • 白色量が約1.6倍ずつ変化する、a, c, e, g, i, l, n, p の8段階（あせぎるNP）。
    • a: 最も明るい白（白色量 89）
    • p: 最も暗い黒（白色量 3.5）

等色相三角形と系列

  • 東白系列: 白色量($W$)が等しい色のグループ。三角形の底辺（$p-C$）に平行。
  • 等黒系列: 黒色量($B$)が等しい色のグループ。三角形の上辺（$a-C$）に平行。
  • 等純系列 (シャドウシリーズ): 純色量($C$)と白色量($W$)の比が等しい。縦軸（$a-p$）に平行。
  • 等価値色系列: 異なる色相間で、記号（白色量・黒色量）が同じ色のこと。

5. NCS (Natural Color System)

1979年にスウェーデンで発表。物理的な混合ではなく、「心理的な類似度」を基準としています。

ヘリングの6主要色

理想的な 白(w), 黒(s), 黄(y), 赤(r), 青(b), 緑(g) を基準とし、これらへの類似度（％）で色を表します。
* $w + s + y + r + b + g = 100$

色相の表記

心理四原色間の位置を類似度で示します。
* 例: Y50R（黄と赤の中間＝オレンジ）
* 例: B10G（青に近く、緑が10%混ざった色）

色記号の読み方

  • 例：S 1030-R10B
    • S: 標準(Standard)
    • 10: 黒色度 ($s$) = 黒への類似度 10%
    • 30: クロマチックネス ($c$) = 純色への類似度 30%
    • （※白色度 $w$ は $100 - (10+30) = 60$% と導き出せる）
    • R10B: 色相（赤に青が10%混ざった色）

6. 色彩語と基本色彩語 (Basic Color Terms)

色空間は連続体ですが、人間はそれを特定のカテゴリーに区切って認識します。

バーリンとケイの理論 (1969)

世界中の言語を調査した結果、色彩語の分化には一定の法則性（進化の順序）があることが発見されました。
* 第1段階: 白・黒（明・暗）
* 第2段階: 赤
* 第3・4段階: 黄、緑（順不同）
* 第5段階: 青
* 第6段階: 茶
* 第7段階: 紫、ピンク、オレンジ、灰色
* 最終的に11種類の基本色彩語に集約されます。

ヘリングの理論との関連

基本色彩語は、ヘリングの「心理四原色（赤・黄・緑・青）」＋「白・黒」の6主要色と深く関連しており、言葉を覚える前の乳幼児でも識別可能とされています。

7. ISCC-NBS色名法

アメリカで制定された、マンセル色立体と対応する組織的な色名表示法です。
* 6つのレベル: レベル1（最も大まか）からレベル6（厳密な数値）まで、用途に応じて精度を使い分けます。
* レベル3が、日本のJISでいう「系統色名」に相当します。

8. JISの色名規格

JIS Z 8102（物体色の色名）および JIS Z 8110（光源色の色名）による分類。

色名の分類

  1. 基本色名: 無彩色3色、有彩色10色の計13色。
  2. 系統色名: 修飾語 ＋ 基本色名 で表す（例：鮮やかな黄みの赤）。
  3. 固有色名: 特定の物に由来する名前（例：桜色）。
    • 慣用色名: 固有色名のうち、日常的に広く使われるもの。

光源色の系統色名

物体色と異なり、「明度に関する修飾語」がありません。
* 修飾語: 「薄い(pl.)」「鮮やかな(v.)」の2種類のみ。
* 基本色名: マンセル10色 ＋ ピンク の11色。
* 語順: [鮮やかさ] ＋ [色相] ＋ [基本色名]（例：薄い 紫みの ピンク）
