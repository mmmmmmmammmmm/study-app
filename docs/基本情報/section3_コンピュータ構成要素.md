# 【Section 3】コンピュータ構成要素

## 1. プロセッサ (CPU)
CPUはコンピュータの頭脳であり、「演算装置」と「制御装置」から構成されます。
*   **命令実行サイクル**: メモリからの「命令フェッチ（読出し）」→「デコード（解読）」→「オペランド（データ）読出し」→「実行」→「ライトバック（書き込み）」という一連のサイクルを繰り返します。
*   **パイプライン制御**: 複数の命令を、1ステージずつずらしながら工場のように並列実行し、処理効率（スループット）を向上させる技術です。条件分岐によってパイプラインが止まる（ハザード）のを防ぐため「分岐予測」が行われます。
*   **性能指標**: クロック周波数（1秒間のクロック信号の回数：Hz）、MIPS（1秒間に実行できる命令を100万単位で表す）。

## 2. メモリと階層構造
CPUの超高速な処理速度と、記憶装置の遅いアクセス速度の差を埋めるため、メモリは階層化されています。
*   **階層**: レジスタ（CPU内・最速） ＞ キャッシュメモリ（SRAM） ＞ 主記憶装置（DRAM） ＞ 補助記憶装置（HDD/SSD）。
*   **キャッシュメモリ**: よく使うデータを主記憶から写しておく高速メモリです。「ヒット率（探しているデータがキャッシュにある確率）」を用いた実効アクセス時間の計算が重要です。
*   **メモリインターリーブ**: 主記憶を複数のバンクに分割し、連続するアドレスに対して並列にアクセスすることで読み書きを高速化する技術です。
