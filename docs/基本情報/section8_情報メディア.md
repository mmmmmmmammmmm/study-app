# 【Section 8】情報メディア

## 1. マルチメディア技術と画像フォーマット
文字、音声、静止画、動画など、多様なメディアを統合して扱う技術です。
*   **JPEG**: 写真などフルカラーで滑らかな階調の画像に適した「非可逆圧縮（完全に元に戻せないが圧縮率が高い）」フォーマットです。
*   **GIF**: 256色までしか扱えない代わりに「可逆圧縮（元のデータを完全に復元できる）」を採用し、イラストやロゴ、アニメーション、透過画像に適しています。
*   **PNG**: GIFに代わる次世代規格として開発され、フルカラーを扱える「可逆圧縮」フォーマットです。Webで広く利用されます。
*   **SVG**: ドットの集まり（ラスタ画像）ではなく、線や曲線の数式データ（ベクタ画像）として画像を表現します。拡大・縮小しても画質が全く荒れない（ジャギーが出ない）のが特徴です。

## 2. マルチメディアの応用
*   **VR (Virtual Reality: 仮想現実)**: HMD（ヘッドマウントディスプレイ）などを装着し、コンピュータが作り出した3DCGの仮想空間の中に、あたかも現実に入り込んだかのような没入感を与える技術です。
*   **AR (Augmented Reality: 拡張現実)**: スマートフォンのカメラ越しに見る「現実の風景」に、CGのキャラクターやナビゲーション情報などの「デジタル情報」を重ね合わせて表示し、現実世界を拡張する技術です（例：ポケモンGO）。
