# 【Section 22】企業活動

## 1. 企業組織
* **機能別組織**: 営業、製造、総務など、専門的な機能ごとに編成された一般的な組織形態。専門性は高まるが、部門間の壁（セクショナリズム）ができやすい。
* **事業部制組織**: 製品別、地域別などに「事業部」を設け、各事業部に利益責任を持たせる組織形態。
* **マトリックス組織**: 機能別と事業部別など、2つの指揮命令系統を交差（マトリックス）させた組織。柔軟な対応が可能だが、命令系統が複雑になる。

## 2. 会計・財務
* **貸借対照表 (B/S)**: ある時点における企業の財政状態（資産、負債、純資産）を示す表。左側に資産、右側に負債と純資産（資本）が並ぶ。
* **損益計算書 (P/L)**: 一定期間における企業の経営成績（収益、費用、利益）を示す表。
* **損益分岐点**: 売上高と総費用（固定費＋変動費）が等しくなり、利益がプラスマイナスゼロになる売上高。これを超えると利益が出る。
