# 【Section 20】技術戦略マネジメント

## 1. 技術開発戦略
自社の競争優位を確立するために、どの技術に投資し、どのように技術を育てていくかの戦略です。
* **MOT (Management of Technology: 技術経営)**: 技術革新をビジネスに結びつけ、経済的価値を創出していくための経営手法。
* **イノベーションのジレンマ**: 大企業が既存の優れた製品（持続的イノベーション）の改良に注力するあまり、顧客の別のニーズに応える新技術（破壊的イノベーション）の登場に遅れをとり、新興企業に市場を奪われてしまう現象。
* **死の谷 (Valley of Death)**: 基礎研究の成果が、製品化・事業化に至るまでの間にある、資金やリソースの不足という大きな障壁。
* **プロダクトライフサイクル**: 製品が市場に投入されてから姿を消すまでの過程（導入期 → 成長期 → 成熟期 → 衰退期）。
