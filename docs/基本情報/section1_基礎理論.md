# 【Section 1】基礎理論

## 1. 離散数学
コンピュータはすべてのデータを0と1のビットで処理します。このデジタルな世界を数学的に扱うのが離散数学です。
*   **基数変換**: 日常で用いる10進数を、コンピュータ内部の2進数（0と1）、プログラムで扱いやすい16進数（0〜9とA〜F）へ変換する仕組みを理解することが不可欠です。
*   **負の数の表現（2の補数）**: コンピュータは引き算を加算回路で処理するため、ビットを反転（1の補数）させて1を足す「2の補数表現」を用います。
*   **論理演算**: AND（論理積）、OR（論理和）、NOT（論理否定）、XOR（排他的論理和）の真理値表。ド・モルガンの法則により論理式を簡略化し、論理回路を最適化します。

## 2. 情報理論と通信理論
*   **情報量**: 情報の持つ不確実さ（エントロピー）をビット単位で表します。シャノンの定理が基礎となります。
*   **誤り検出・訂正**: ネットワーク通信時のノイズによるデータの破損を検知・修復します。
    *   **パリティチェック**: 1ビットの誤り検出のみ可能。
    *   **CRC**: 多項式を用いた精度の高い誤り検出（ネットワークのフレームチェックで利用）。
    *   **ハミング符号**: 2ビットの誤りを検出し、1ビットの誤りを自動訂正できる高度な符号化技術。
