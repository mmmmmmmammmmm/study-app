【Section 7】ユーザーインタフェース

1. ユーザーインタフェース (UI) の基本

コンピュータと人間の接点となる画面や操作方法のことです。
* CUI (Character User Interface): キーボードから文字（コマンド）を入力して操作する方式。バッチ処理（定型処理の自動化）などに適しています。
* GUI (Graphical User Interface): マウスやタッチパネルを使い、画面上のアイコンやボタン（グラフィック）を直感的に操作する方式。

2. UX / UI デザインとユーザビリティ

  • UX (User Experience: ユーザー体験): 製品やサービスを通じてユーザーが得る「楽しさ」「心地よさ」「満足感」など、体験全体の質を指します。
  • ユーザビリティ (使いやすさ): ISO規格において「有効性（目標を達成できるか）」「効率性（無駄なくできるか）」「満足度」の3つの要素で定義されています。
  • アクセシビリティ: 高齢者や視覚・聴覚に障害を持つ人を含め、「誰もが」システムを不自由なく利用できること。読み上げ機能や、色覚多様性に配慮したカラーユニバーサルデザイン(CUD)などが含まれます。
  • フールプルーフ: ユーザーが誤った操作（Fool）をしても、システムが危険な状態にならない（エラーを未然に防ぐ）ように設計する安全思想です。
