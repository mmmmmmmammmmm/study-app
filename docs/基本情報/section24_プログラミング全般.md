# 【Section 24】プログラミング全般

## 1. プログラミング (科目B対策)
基本情報技術者試験の「科目B」では、特定のプログラミング言語（C, Java, Pythonなど）の構文ではなく、共通の「擬似言語」を用いたアルゴリズムの読解問題が出題されます。

## 2. 擬似言語の基本構造
* **順次処理**: 上から下へ順番に処理を実行する。
* **選択処理 (条件分岐)**: `if (条件)` によって処理を分ける。
* **繰返し処理 (ループ)**: `for` や `while` によって、特定の条件を満たす間、処理を繰り返す。

## 3. 重要テーマ
科目Bのプログラミングで頻出となるアルゴリズムや考え方は以下の通りです。
* **配列の走査と集計**: 配列の全要素をループで確認し、合計、平均、最大値、最小値、条件に合致する要素のカウントを行う。
* **文字列処理**: 文字列の検索、置換、反転などの操作。
* **再帰処理**: 関数の中で、自分自身の関数を呼び出す処理。階乗の計算や、木構造の探索（深さ優先探索など）で用いられる。無限ループにならないよう「終了条件」が必須。
* **データ構造の操作**: スタックへのPush/Pop、キューへのEnqueue/Dequeue、連結リストのノード挿入・削除のポインタ操作のトレース。

## 4. トレース能力の向上
問題を解くためには、変数の値がループごとにどのように変化していくかを、紙に書き出して正確に追跡する（トレースする）能力が求められます。
