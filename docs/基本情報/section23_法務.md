# 【Section 23】法務

## 1. 知的財産権
* **著作権**: 思想又は感情を創作的に表現した「著作物」を保護する権利。ソフトウェアの場合、ソースコードやマニュアルは保護されるが、**「アルゴリズム」「プログラミング言語」「規約（プロトコル）」は保護対象外**である点に注意。
* **特許権**: 産業上利用できる新規の発明を保護する。ビジネスモデル特許などもある。

## 2. 労働関連・取引関連法規
* **労働者派遣法**: 派遣元企業から派遣先企業へ労働者を派遣する際のルール。派遣先からさらに別の企業へ派遣する「二重派遣」は禁止されている。
* **請負契約と準委任契約**: 外部に業務を委託する際の契約形態。
    * **請負契約**: 「仕事の完成（成果物の納品）」に対して報酬が支払われる。瑕疵担保責任（契約不適合責任）がある。
    * **準委任契約**: 「専門的な業務の遂行」自体に対して報酬が支払われる。仕事の完成義務はないが、善管注意義務を負う。

## 3. その他の法律
* **不正アクセス禁止法**: 他人のID/パスワードの不正利用や、セキュリティホールを突いた不正侵入を禁止する。
* **個人情報保護法**: 生存する個人の情報を適切に取り扱うためのルール。
