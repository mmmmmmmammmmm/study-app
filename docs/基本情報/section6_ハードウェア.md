# 【Section 6】ハードウェア

## 1. 電気・電子回路
*   **A/D変換 (アナログ-デジタル変換)**: 音声などの連続したアナログ信号を、コンピュータで扱えるデジタル信号に変換するプロセスです。
    1.  **標本化 (サンプリング)**: 一定の時間間隔で信号の値を読み取ります（サンプリング定理）。
    2.  **量子化**: 読み取った値を離散的な（飛び飛びの）数値に丸めます。
    3.  **符号化**: 量子化された数値を2進数のビット列に変換します。

## 2. 半導体素子と論理設計
*   **システムLSI と SoC (System on a Chip)**: マイコン、メモリ、通信回路など、システムに必要なすべての機能を1つの半導体チップ上に集積したもの。スマートフォンなどに用いられ、小型化と省電力化に寄与します。
*   **FPGA (Field-Programmable Gate Array)**: 工場での製造後であっても、開発者が論理回路の構成を自由にプログラミング（書き換え）できる半導体チップです。
*   **MEMS (Micro Electro Mechanical Systems)**: 半導体微細加工技術を用いて、センサー（加速度センサーなど）やアクチュエータといった機械的な要素と電子回路を1つのシリコン基板上に組み込んだ微小デバイスです。
