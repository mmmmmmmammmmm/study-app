【Section 10】環境と色彩（アドバンス色彩対応）

建築物や公共空間における色彩は、個人の好みを超えて「公共財」としての側面を持ちます。景観形成の歴史と、環境色彩計画のプロセスについて学びます。

1. 建築物等の色彩計画の歴史と法律

かつての高度経済成長期、日本の都市は派手な看板や建物で溢れ、景観の乱れが問題となりました。
* 景観法（2004年制定）: 日本で初めて景観について総合的に定めた法律です。自治体が「景観計画」を策定し、建物の色などを規制・誘導できるようになりました。
* 色彩ガイドライン: 多くの自治体では、マンセル値（色相・明度・彩度の範囲）を用いて、周囲に調和する色の範囲を指定しています。

2. 環境色彩計画の手順

プロのカラーコーディネーターが行う環境色彩計画は、以下のステップで進められます。
1. 現況調査: 周辺環境の色（自然の色、既存建物の色）を測色計や色票で調査します。
2. コンセプト立案: 地域の歴史や機能に合わせ、どのような景観を目指すか決定します。
3. 色彩設計: ベースカラー、アソートカラー、アクセントカラーを決定し、シミュレーションを行います。
4. 検証・実施: 実際の素材（タイル、塗装板など）を現地の光の下で確認（サンプリング）し、最終決定します。

3. これからのカラーコーディネーターの役割

持続可能な社会（SDGs）において、色彩は単なる装飾ではなく、省エネルギー（遮熱塗料の色選びなど）、安全性の向上、メンタルヘルス（癒やしの空間づくり）など、多岐にわたる課題解決の手段として期待されています。

演習問題（理解度チェック）

Q1. 日本において、良好な景観の形成を目的として2004年に制定された法律はどれか。
1. 都市計画法
2. 景観法
3. 建築基準法
4. 環境基本法

Q2. 環境色彩計画において、建物に使用する色を決定する際、最終的な確認として最も重要なプロセスはどれか。
1. パソコンの画面上だけでシミュレーションする。
2. 室内（蛍光灯下）で色見本を確認する。
3. 実際の素材サンプルを、現地の屋外の光の下で確認する。
4. 過去の成功事例と全く同じ色を採用する。

解答と解説

  • Q1. 解答：2
    • 解説：景観法により、色彩規制が法的根拠を持つようになりました。
  • Q2. 解答：3
    • 解説：色は光源（太陽光）や周囲の環境に大きく左右されるため、現地での現物確認（サンプリング）が不可欠です。
