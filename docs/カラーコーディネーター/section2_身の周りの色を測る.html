【Section 2】身の周りの色を測る（アドバンスクラス対応）

カラーコーディネーターの実務において、色を「感覚」ではなく「数値（尺度）」として客観的に捉えることは不可欠です。本章では、測色の基礎理論から、ビジネス現場で用いられる具体的な測定尺度について学習します。

1. 測るとは何か―ものさし（尺度）の性質

「測る」とは、ある対象の属性を数値や記号で表すことです。心理学や統計学では、以下の4つの尺度が重要です。
* 名義尺度: 性別や血液型のように、分類のみを目的とする。
* 順序尺度: 好みの順位など、大小関係はあるが間隔が等しくない。
* 間隔尺度: 温度（℃）のように、間隔は等しいが「ゼロ」が絶対的ではない。
* 比率尺度: 長さや重さのように、間隔が等しく、かつ「ゼロ」が「存在しないこと」を意味する。色彩の物理的な測定値（反射率など）はこれにあたります。

2. 色を測るときの尺度と尺度の変換

色は「三属性（色相・明度・彩度）」という心理的な尺度と、「刺激値（X, Y, Z）」という物理的な尺度で表されます。
* 心理物理的尺度: 人間の視覚特性を考慮して物理量を変換したもの。CIE（国際照明委員会）が規定するXYZ表色系が代表的です。
* 一対比較法・順位法: 人間の感覚（どちらがより赤く見えるか等）を数値化する手法。

3. 色を測る（測色）

物体色を測定する方法には、大きく分けて「視感測色」と「物理測色」があります。
* 視感測色: 標準の光の下で、色見本（マンセル色票など）と目視で比較する方法。熟練を要しますが、人間の感覚に近いです。
* 物理測色（計器測色）:
* 分光測色法: 分光測色計（スペクトロフォトメーター）を用い、波長ごとの反射率を測定する最も精度の高い方法。
* 刺激値直読法: 色彩計（カラーメーター）を用い、人間の目に対応する3つのフィルターを通してXYZ刺激値を直接求める方法。簡易的で現場向きです。
* 色差（ΔE）: 2つの色の違いを数値化したもの。Lab*色空間（エル・スター、エー・スター、ビー・スター）において、2点間の距離として計算されます。

演習問題（理解度チェック）

Q1. 最も精度の高い測色方法であり、各波長ごとの分光反射率を測定して色を数値化する手法はどれか。
1. 順位法
2. 刺激値直読法
3. 分光測色法
4. 一対比較法

Q2. CIE Lab*色空間における「色差」を表す記号として正しいものはどれか。
1. ΔH
2. ΔE
3. ΔL
4. ΔC

解答と解説

  • Q1. 解答：3
    • 解説：分光測色計を用いた分光測色法が、最も情報量が多く高精度な測定方法です。
  • Q2. 解答：2
    • 解説：色差は「デルタ・イー（ΔE）」で表されます。数値が小さいほど色が近く、大きいほど色の差が激しいことを示します。
