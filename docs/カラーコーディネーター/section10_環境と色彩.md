# 【Section 10】環境と色彩

## 1. 景観色彩とガイドライン
街並みや建築物の色は、周囲の自然や歴史的文脈と調和している必要があります。日本では2004年に「景観法」が制定され、多くの自治体が独自の色彩ガイドライン（使用できるマンセル値の制限、彩度の上限など）を定めて、乱雑な色彩を規制しています。

## 2. サイン計画 (ピクトグラムと安全色)
公共の場（駅、空港、病院など）において、案内や警告を直感的に伝える「サイン」には、高い視認性と誘目性が求められます。言葉の壁を越えるピクトグラム（図記号）とともに、JIS（日本産業規格）で定められた安全色が厳密に使用されます。
*   **赤**: 防火、禁止、停止
*   **黄**: 警告、注意
*   **緑**: 安全、避難、衛生
*   **青**: 指示、義務

## 3. 光害 (ひかりがい)
過剰な夜間の照明やネオンサインが、動植物の生態系を狂わせたり、天体観測を阻害したりする問題を「光害」と呼びます。環境色彩においては、色だけでなく「光の量と質」も適切にコントロールする必要があります。
