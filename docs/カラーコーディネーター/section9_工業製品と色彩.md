# 【Section 9】工業製品と色彩

## 1. プロダクトデザインにおけるCMF
家電、スマートフォン、自動車などの工業製品において、色は製品のコンセプトを伝え、ターゲット層に訴求する重要な要素です。「CMFデザイン」と呼ばれ、**Color（色）、Material（素材）、Finish（仕上げ）**の3つを総合的に設計します。

## 2. 表面仕上げの効果
同じ赤いプラスチックの樹脂でも、表面を「鏡面仕上げ（ツヤあり、グロス）」にするか、「シボ加工（ツヤ消し、マット）」にするかで、見え方（明度や彩度）や受ける印象（高級感、親しみやすさなど）が大きく変わります。光の乱反射の度合いが異なるためです。

## 3. 自動車のボディカラー
自動車の色は、国や地域の気候、文化、トレンドに大きく左右されます。シルバーやホワイトなどの無彩色は汚れが目立ちにくくリセールバリューが高いため世界的に人気ですが、スポーティな車種では彩度の高い赤や黄色も選ばれます。塗装技術の進歩により、光の当たる角度で色が変わるメタリックパール塗装なども普及しています。
