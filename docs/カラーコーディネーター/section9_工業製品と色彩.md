# 【Section 9】工業製品と色彩（アドバンスクラス対応）

工業製品（プロダクト）における色彩は、機能の伝達、ブランドイメージの確立、そしてユーザーの所有欲を喚起する重要な要素です。近現代のデザイン史とCMFデザインの考え方について学びます。

---

## 1. 近現代のデザインとカラー
工業製品の色彩は、技術革新と社会情勢に合わせて変遷してきました。
*   **バウハウス**: 「形態は機能に従う」を掲げ、基本形状と原色を重視した近代デザインの礎を築きました。
*   **1950年代（アメリカ）**: 大量生産・大量消費時代。カラフルな家電や自動車（テールフィンなど）が登場し、色の流行が意識されるようになりました。
*   **ユニバーサルデザインの台頭**: 誰もが使いやすい操作パネルの配色など、機能美とアクセシビリティの両立が求められるようになりました。

## 2. 商品のカラーデザインとCMF
現代のプロダクトデザインでは、色単体ではなく「CMF」という概念で設計が行われます。
*   **Color（色）**: 色相、明度、彩度の選定。
*   **Material（素材）**: プラスチック、金属、木材、ファブリックなど、素材自体の色と質感。
*   **Finish（仕上げ）**: 表面処理。鏡面仕上げ（グロス）、梨地仕上げ（マット）、ヘアライン加工など。
これら3要素を組み合わせることで、同じ「黒」でも「高級感のある黒」「プロフェッショナルな黒」「親しみやすい黒」を使い分けます。

## 3. 自動車のデザインと時代変化
自動車は工業製品のなかでも特にCMFの結晶と言えます。
*   **ボディカラー**: 塗料技術（メタリック、パール、マイカなど）により、複雑な曲面を強調し、光の当たり方で表情を変える設計がなされます。
*   **リセールバリュー**: 白、シルバー、黒などの無彩色は流行に左右されにくく、中古車市場での価値が高いため、ビジネス的な観点からも選ばれやすい特徴があります。

## 演習問題（理解度チェック）

**Q1.** プロダクトデザインにおいて、色、素材、仕上げを統合的に設計する考え方を指すアルファベット3文字の用語はどれか。
1. CMS
2. CMF
3. CUD
4. RGB

**Q2.** 金属の表面に細かな線を引くことで、独特の光沢と質感を与える仕上げ加工を何と呼ぶか。
1. シボ加工
2. 鏡面仕上げ
3. ヘアライン加工
4. 梨地仕上げ

---

### 解答と解説

*   **Q1. 解答：2**
    *   解説：Color, Material, Finishの略です。
*   **Q2. 解答：3**
    *   解説：髪の毛のような細い線を引く加工で、高級家電や時計などに多用されます。
