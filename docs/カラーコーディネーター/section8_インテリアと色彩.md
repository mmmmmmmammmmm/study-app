# 【Section 8】インテリアと色彩

## 1. 室内空間の色彩設計
インテリアの配色は、面積比によって大きく3つに分けられます。
*   **ベースカラー (基調色: 70%)**: 壁、床、天井など。空間の雰囲気を決定づけるため、明るく刺激の少ない色（オフホワイトやベージュなど）が好まれます。
*   **アソートカラー (配合色: 25%)**: カーテン、ソファ、ラグ、建具など。ベースカラーと調和し、インテリアの主役となる色。
*   **アクセントカラー (強調色: 5%)**: クッション、絵画、小物など。空間を引き締める高彩度な色。

## 2. 部屋の用途と色彩心理
寝室には鎮静効果や睡眠導入効果のある青や低彩度色（寒色系やアースカラー）が適しています。一方、ダイニングには食欲を増進させる暖色系（オレンジや黄色）や、料理を美味しく見せる演色性の高い照明（電球色）が求められます。

## 3. 錯視の利用
天井を床よりも明るい色にすると、天井が高く感じられ、部屋が広く見えます。逆に天井を暗くすると、空間に落ち着きや圧迫感が生まれます。
