# 【Section 6】ファッションと色彩

## 1. ファッションにおける色の役割
ファッションにおいて色は、着る人のパーソナリティを表現し、季節感やトレンドを伝える最も重要な要素です。毎年「インターカラー（国際流行色委員会）」によって、実シーズンの約2年前にトレンドカラーが選定・提案され、そこから素材や製品の企画が始まります。

## 2. パーソナルカラー
その人が生まれ持った肌、瞳、髪の色に調和し、魅力を引き立てる色を「パーソナルカラー」と呼びます。大きく「イエローベース（春・秋）」と「ブルーベース（夏・冬）」の4シーズンに分類するのが一般的です。

## 3. アパレルVMD (ビジュアル・マーチャンダイジング)
店舗において、商品を魅力的に見せ、購買意欲を高めるための視覚的な演出です。同系色でまとめたり、補色を使ってアクセントをつけたりするカラー展開（カラープレゼンテーション）が売上を大きく左右します。
