# 【Section 3】色彩調和と配色（アドバンスクラス対応）

配色デザインの良し悪しを判断する基準となる「色彩調和論」の歴史的背景と、現代のビジネスで活用される具体的な配色構成の原理について深く学びます。

---

## 1. 配色の構成要素
配色とは、2色以上の色を組み合わせ、特定の効果や美しさを生み出すことです。構成要素として、**「主役の色（ベースカラー）」「引き立てる色（アソートカラー）」「強調する色（アクセントカラー）」**の面積比（例えば 70:25:5）が重要となります。

## 2. 配色構成のバランス・リズム・変化と統一
美しい配色には、音楽や建築と同様の造形原理が働いています。
*   **バランス（均衡）**: 左右対称（シンメトリー）や非対称（アシンメトリー）による安定感。
*   **リズム（律動）**: 色を段階的に変化させる「グラデーション」や、一定の間隔で繰り返す「レピティション」。
*   **統一と変化**: 全体のトーンを揃えて「統一感」を出しつつ、一部に反対色を入れて「変化」をつけることで、退屈さを回避します。

## 3. 調和の原理（ジャッドの4つの原則）
アメリカの色彩学者デビッド・ジャッド（Deane B. Judd）は、過去の多くの調和論を4つの原則に集約しました。
1.  **秩序の原理**: 色相環上で規則的な位置（等間隔など）にある色は調和する。
2.  **親近性（なじみ）の原理**: 自然界で見慣れた色の組み合わせ（例：夕焼けのグラデーション）は調和する。
3.  **共通性（類似性）の原理**: 何か共通の属性（同じ色相、同じトーンなど）を持つ色は調和する。
4.  **明瞭性（明白性）の原理**: 2色の差がはっきりしているか、逆に極めて近い場合は調和する（中途半端な差は不快感を与える）。

## 4. 日本の伝統色の特徴
日本の伝統色は、自然の動植物や鉱物、染料の名前に由来するものが多く（例：桜色、群青色）、季節感を重視するのが特徴です。また、微妙な中間色（四十八茶百鼠）を愛でる文化も、日本の配色感覚の根底にあります。

## 演習問題（理解度チェック）

**Q1.** ジャッドの調和原則のうち、「共通の属性を持つ色は調和する」とするものはどれか。
1. 秩序の原理
2. 親近性の原理
3. 共通性の原理
4. 明瞭性の原理

**Q2.** 配色の構成比率として一般的に推奨される、ベースカラー、アソートカラー、アクセントカラーの比率（%）の組み合わせはどれか。
1. 50：40：10
2. 70：25：5
3. 33：33：33
4. 90：5：5

---

### 解答と解説

*   **Q1. 解答：3**
    *   解説：類似性の原理とも呼ばれます。
*   **Q2. 解答：2**
    *   解説：黄金比に近い 70:25:5 が、最もバランスが良い配分の目安とされています。
