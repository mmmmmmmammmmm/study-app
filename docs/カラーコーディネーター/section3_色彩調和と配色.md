# 【Section 3】色彩調和と配色

## 1. 色彩調和論の歴史
歴史上、多くの研究者が「どのような色の組み合わせが美しく調和するか」を研究してきました。ジャッドの4つの調和原則（秩序の原則、なじみの原則、類似性の原則、明瞭性の原則）が特に有名です。

## 2. ジャッドの4つの調和原則
*   **秩序の原則**: 規則的に選ばれた色（マンセル色相環上で等間隔など）は調和する。
*   **なじみの原則**: 自然界の風景など、見慣れた色の組み合わせは調和する。
*   **類似性の原則**: 共通の属性（色相が同じ、明度が同じなど）を持つ色は調和する。
*   **明瞭性の原則**: 色の差が曖昧ではなく、はっきりと区別できる（コントラストがある）色は調和する。

## 3. 代表的な配色技法
*   **ドミナントカラー配色**: 全体を同じ色相で統一する配色。
*   **トーンオントーン配色**: 同一色相で、明度差（トーンの差）を大きくつけた配色。
*   **コンプレックス配色**: 自然界の法則（明るい色が黄み、暗い色が青み）の逆をいく、人工的で前衛的な配色。
