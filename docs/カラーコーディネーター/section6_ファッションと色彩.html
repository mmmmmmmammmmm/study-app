【Section 6】ファッションと色彩（アドバンスクラス対応）

ファッションにおける色彩は、自己表現の手段であると同時に、トレンド（流行）や購買心理に強く支配されるビジネス領域です。ファッションカラーの特質と、代表的な配色技法について学びます。

1. ファッションカラーの特質

ファッションにおける色は、素材（布、革、ニットなど）の質感やドレープ（陰影）と切り離せません。
* トレンドカラー（流行色）: 国際流行色委員会（インターカラー）によって約2年前から検討され、実シーズンに市場へ投入されます。
* パーソナルカラー: 個人の肌、目、髪の色に調和する色のこと。「春・夏・秋・冬」の4シーズンに分類する手法が一般的です。

2. ファッションに多用される配色技法

  • トーン・オン・トーン配色: 同一色相で、明度差を大きくつけた配色（例：ライトブルーとネイビー）。「色の上に色を重ねる」という意味で、統一感が出やすいです。
  • トーン・イン・トーン配色: 異なる色相で、トーン（明度・彩度のニュアンス）を揃えた配色（例：パステルピンクとパステルブルー）。全体の雰囲気を統一するのに有効です。
  • カマイユ配色: ほとんど同じ色に見える、極めて微妙な差の配色。さらに差をつけたものを「フォ・カマイユ」と呼びます。
  • トリコロール配色: フランス国旗のように、明快な3色（通常は白を含む）を組み合わせた、コントラストの強い配色。

3. ファッションビジネスと色彩管理

アパレル製品は、企画から生産、店頭に並ぶまで多くの工程を経るため、正確な「色合わせ」が重要です。標準光の下での目視確認や、数値による色差管理（ΔE）が実務で行われます。

演習問題（理解度チェック）

Q1. 同一色相で明度差を大きくつけた配色の名称として正しいものはどれか。
1. トーン・イン・トーン配色
2. トーン・オン・トーン配色
3. カマイユ配色
4. トリコロール配色

Q2. 「ほとんど同じ色に見える、極めて微妙な差の配色」を指す用語はどれか。
1. ビコロール
2. フォ・カマイユ
3. カマイユ
4. コンプレックス

解答と解説

  • Q1. 解答：2
    • 解説：同系色の濃淡でまとめる、ファッションの王道配色です。
  • Q2. 解答：3
    • 解説：カマイユは「単色画」の意味で、遠目には1色に見えるほど近い色同士の調和を指します。
