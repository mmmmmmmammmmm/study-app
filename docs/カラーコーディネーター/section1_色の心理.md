# 【Section 1】色の心理

## 1. 色彩心理の基礎
色は人間の心理や生理に大きな影響を与えます。例えば、赤は交感神経を刺激して脈拍を上げ、暖かさを感じさせる「暖色」の代表です。一方、青は副交感神経を優位にし、リラックスや冷たさを感じさせる「寒色」です。これにより、体感温度は赤と青で約3度異なるとも言われます。

## 2. カラーマーケティングと色の効果
企業はロゴやパッケージにコーポレートカラーを用い、消費者の購買意欲をコントロールします。例えば、ファストフード店に赤や黄色が多いのは、食欲を増進させ、客の回転率を上げる（時間を長く感じさせる）効果があるためです。

## 3. 共感覚 (Synesthesia)
ある特定の刺激から、通常の感覚に加えて別の感覚も引き起こされる現象を「共感覚」と呼びます。「音を聴くと色が見える（色聴）」「文字に色がついて見える」などが代表的です。色彩心理学において、色が持つイメージ（重い、甘いなど）の多くはこの共感覚的な連想から生じます。
