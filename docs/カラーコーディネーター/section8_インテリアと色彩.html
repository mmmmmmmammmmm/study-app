【Section 8】インテリアと色彩（アドバンスクラス対応）

インテリアにおける色彩は、空間の広さの感じ方（容積感）、温冷感、そして居住者の活動内容に合わせた機能性が求められます。

1. インテリアカラーの構成

インテリアの配色は、面積比によって以下の3つに大別されます。
1. ベースカラー（基調色）: 壁、床、天井など（約70%）。空間の印象を決定づけます。
2. アソートカラー（配合色）: 家具、カーテン、ドアなど（約25%）。ベースカラーと調和させます。
3. アクセントカラー（強調色）: クッション、絵画、小物など（約5%）。変化と活気を与えます。

2. 空間を演出する視覚効果

  • 容積感（広さ）: 明度の高い色（白系）は膨張して見えるため、部屋を広く感じさせます。逆に暗い色は収縮し、圧迫感を与えます。
  • 天井高: 天井を壁よりも明るくすると高く感じ、暗くすると低く落ち着いた雰囲気になります。
  • 温冷感: リビングにはあたたかみのある暖色系、寝室や書斎には落ち着く寒色系や中性色系が選ばれることが多いです。

3. インテリアデザインのスタイルとカラー

  • ナチュラル: ベージュ、ブラウン、グリーンなどのアースカラー。
  • モダン: 白、黒、グレーのモノトーンを基調とした、コントラストの強い配色。
  • 北欧: 明るい木の色に、彩度の高いアクセントカラー（黄色や青）を効かせた配色。

演習問題（理解度チェック）

Q1. インテリアの配色において、壁や床などの大きな面積を占める「ベースカラー」が占める割合の目安はどれか。
1. 約10%
2. 約25%
3. 約70%
4. 約95%

Q2. 部屋を広く開放的に見せるための色の組み合わせとして、最も適切なものはどれか。
1. 壁と天井を、低明度の暗い色にする。
2. 壁と天井を、高明度の明るい色にする。
3. 床を白、壁を黒にする。
4. 全体を高彩度の鮮やかな色にする。

解答と解説

  • Q1. 解答：3
    • 解説：ベースカラーは空間の背景となるため、7割程度を占めます。
  • Q2. 解答：2
    • 解説：高明度の色は「膨張色」であり、壁面が外側に広がって見えるため、広さを感じさせます。
