# 【Section 7】メイクアップ化粧品と色彩

## 1. メイクアップの色彩学
メイクアップは、顔の立体感を強調したり、血色を良く見せたりするために色彩の錯視効果を最大限に利用します。例えば、ハイライト（高明度）で高さを出し、シェーディング（低明度）で影を作って小顔効果を演出します。これを「コントゥアリング」と呼びます。

## 2. 肌色とファンデーション
肌の色はメラニン色素（茶〜黒）、酸化ヘモグロビン（赤）、カロチン（黄）のバランスで決まります。ファンデーション選びでは、自分の首（あるいはフェイスライン）の色と自然に馴染む色相と明度を選ぶことが重要です。

## 3. 色の対比現象の応用
アイシャドウやリップカラーは、周囲の肌色との対比（明度対比や彩度対比）によって見え方が変化します。例えば、青みの強いピンクのリップを塗ると、相対的に肌が黄みがかって（あるいは白く透明感があるように）見える効果があります。
