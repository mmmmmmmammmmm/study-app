# 【Section 5】ビジュアルデザインのための色彩技術

## 1. デジタル色彩とカラーマネジメント
パソコンのモニターやスマートフォンはRGB（加法混色）で色を表現し、印刷物はCMYK（減法混色）で色を表現します。デバイス間や、Webと印刷で色が違って見えるのを防ぐ仕組みを「カラーマネジメントシステム (CMS)」と呼びます。

## 2. ICCプロファイル
機器ごとの色の再現特性（ガムット：色再現域）をデータ化したものを「ICCプロファイル」と呼びます。これを画像のデータに埋め込むことで、異なるモニターやプリンターでも、そのプロファイルを参照して可能な限り同じ色を変換・再現できます。

## 3. レタッチと色調補正
Photoshopなどの画像編集ソフトにおいて、トーンカーブやレベル補正を用いて画像の明暗やコントラストを調整したり、特定の色域だけを補正（HSL補正など）して、目的のイメージに近づける技術です。
