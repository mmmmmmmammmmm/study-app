# 【Section 5】ビジュアルデザインのための色彩技術（アドバンスクラス対応）

グラフィック、Web、プレゼンスライドなど、視覚伝達メディアにおいて色彩を戦略的に活用するための技術と、デジタル環境特有のカラーマネジメントについて学習します。

---

## 1. ビジネス実務とビジュアルデザイン
ビジネスにおけるビジュアルデザインの目的は、情報を「正確に」「速く」「魅力的に」伝えることです。カラーコーディネーターは、媒体の特性（紙か画面か）やターゲットに合わせて、最適なカラースキームを提案する役割を担います。

## 2. デジタルメディアの色彩（Web・スライド）
*   **RGBモデル**: コンピュータの画面は、赤(Red)、緑(Green)、青(Blue)の3原色の混色（加法混色）で色を表現します。
*   **Webセーフカラー**: かつての環境で共通して表示できた216色のことですが、現在はフルカラーが主流のため、より自由な配色が可能です。
*   **スライドデザイン**: 背景色と文字色の明度差を大きく取ることが鉄則です（例：濃紺背景に白文字、白背景に黒文字）。また、強調したい箇所には補色（反対色）をアクセントとして使います。

## 3. グラフィックデザイン（印刷）の色彩
*   **CMYKモデル**: 印刷物は、シアン(Cyan)、マゼンタ(Magenta)、イエロー(Yellow)に黒(Key plate)を加えた減法混色で表現されます。
*   **色域（ガマット）**: RGBよりもCMYKの方が表現できる色の範囲が狭いため、画面で見た鮮やかな色が印刷ではくすんでしまう「色域外」の問題に注意が必要です。
*   **カラーマネジメントシステム (CMS)**: モニター、プリンター、印刷機など異なるデバイス間で、色を統一的に管理する技術です。ICCプロファイルという設定ファイルを用いて、色の変換を最適化します。

---

## 演習問題（理解度チェック）

**Q1.** デジタルデバイス（モニターなど）で色を表現する際に用いられる「加法混色」の3原色の組み合わせはどれか。
1. CMY
2. RGB
3. PCCS
4. XYZ

**Q2.** 印刷物の色の再現範囲（ガマット）について、画面表示（RGB）と比較したときの特徴として正しいものはどれか。
1. RGBよりもCMYKの方が表現できる色の範囲が広い。
2. RGBよりもCMYKの方が表現できる色の範囲が狭い。
3. 両者の表現範囲は完全に一致する。
4. 印刷物は光の混色であるため、RGBよりも明るくなる。

---

### 解答と解説

*   **Q1. 解答：2**
    *   解説：Red, Green, Blueの3原色です。
*   **Q2. 解答：2**
    *   解説：CMYKはインクの重なりによる減法混色のため、光の混色であるRGBよりも鮮やかな色（特に高彩度な青や緑）の再現が難しくなります。
