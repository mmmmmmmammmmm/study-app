# 【Section 2】身の周りの色を測る

## 1. 測色の基礎
色を正確に伝達し、品質を管理するためには「測色（色を測ること）」が必要です。人間の眼による「視感測色」と、機械による「物理測色」の2種類があります。

## 2. 物理測色計の種類
*   **分光測色計**: 物体が反射（または透過）する光の波長ごとの強さ（分光反射率）を測定し、正確な色度を算出する高精度な機器です。
*   **刺激値直読測色計**: 人間の眼のセンサー（錐体）に近い感度を持つ3つのフィルター（X, Y, Z）を用いて、直接三刺激値を読み取る簡易的な機器です。

## 3. 視感測色
人間の眼をセンサーとして色を判定する方法です。JISなどで定められた「標準の光源（D65など）」の下で、標準となる色票（マンセル色票など）と、対象物を肉眼で見比べて評価します。評価者の色覚異常の有無や疲労度、年齢による水晶体の黄変などが結果に影響します。
