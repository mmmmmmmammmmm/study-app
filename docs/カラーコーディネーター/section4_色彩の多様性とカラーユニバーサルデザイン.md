# 【Section 4】色彩の多様性とカラーユニバーサルデザイン

## 1. 色覚の多様性
人間の色覚には個人差があり、先天的な遺伝や後天的な眼の疾患（白内障など）、加齢によって、色の見え方は異なります。特に「赤と緑」の区別がつきにくい人が多く存在し、男性の約5%（20人に1人）と言われています。

## 2. カラーユニバーサルデザイン (CUD) の3原則
様々な色覚を持つすべての人にとって、情報が正確に伝わるようにデザインすることをCUDと呼びます。
1.  **できるだけ多くの人に見分けやすい配色を選ぶ。**
2.  **色を見分けにくい人にも情報が伝わるようにする。**（形を変える、文字を併記するなど）
3.  **色の名前を用いたコミュニケーションを可能にする。**（「赤いボタンを押して」ではなく「右の丸いボタンを押して」とするなど）

## 3. シミュレーションと検証
デザインを決定する前に、色覚異常の人の見え方をシミュレートするソフトウェアや専用のフィルター（バリアントールなど）を用いて、問題がないか検証することが重要です。
