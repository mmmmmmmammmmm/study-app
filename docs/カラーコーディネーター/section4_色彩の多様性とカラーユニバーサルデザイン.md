# 【Section 4】色覚の多様性とカラーユニバーサルデザイン（アドバンスクラス対応）

色の見え方は、すべての人にとって同じではありません。遺伝的な特性や加齢、環境によって異なる「多様な見え方」を理解し、すべての人に正確な情報を伝える「カラーユニバーサルデザイン（CUD）」の実践について学びます。

---

## 1. カラーユニバーサルデザイン（CUD）とは
CUDとは、色覚のタイプ（一般色覚、色弱など）を問わず、できるだけ多くの人に情報が正しく伝わるように配慮されたデザインのことです。「バリアフリー」の色彩版と言えます。
*   **3つの基本方針**:
    1.  色だけでなく、形、模様、文字などの情報を併用する。
    2.  明度差（コントラスト）を十分に確保する。
    3.  色の組み合わせを工夫し、識別しやすい色名を用いる。

## 2. 色の見え方の多様性
*   **色覚多様性（色弱・色盲）**: 赤と緑の区別がつきにくい「P型」「D型」が代表的です。日本人男性の約20人に1人、女性の約500人に1人の割合で存在します。
*   **加齢変化**: 年齢を重ねると水晶体が黄色く濁る（黄変化）ため、青系の色が暗く見えたり、黄色と白の区別がつきにくくなったりします。また、白内障による視力低下やコントラスト感度の低下も生じます。

## 3. カラーユニバーサルデザインの活用例
*   **サイン計画**: 避難誘導灯などの安全色彩において、色覚特性に配慮した「ユニバーサルデザイン・カラー」の採用。
*   **ハザードマップ**: 警戒レベルを色だけでなく、ハッチング（網掛け模様）で区別する。
*   **家電製品**: スイッチのON/OFFを赤・緑のランプだけでなく、点灯・点滅やアイコンの併記で伝える。

## 演習問題（理解度チェック）

**Q1.** カラーユニバーサルデザインにおいて、最も避けるべき配慮の欠如はどれか。
1. 色名だけで情報を伝えようとすること。
2. 明度差を大きくつけること。
3. 文字や模様を併用すること。
4. 異なる色覚タイプを想定してシミュレーションを行うこと。

**Q2.** 高齢者の色の見え方の特徴として、正しいものはどれか。
1. 赤い色が全く見えなくなる。
2. 水晶体が黄変化し、青や紫が識別しにくくなる。
3. 暗い場所での感度が若者よりも向上する。
4. 明度差が小さくても形を正確に把握できるようになる。

---

### 解答と解説

*   **Q1. 解答：1**
    *   解説：色覚多様性を持つ方にとって、色だけで情報を区別するのは困難な場合があります。
*   **Q2. 解答：2**
    *   解説：水晶体の黄変化により、短波長の光（青・紫）が透過しにくくなります。
